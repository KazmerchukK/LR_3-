Java
Означення - Java - це об'єктно-орієнтована мова програмування, яка використовується для створення програмного забезпечення різних типів, від мобільних 
додатків до корпоративних систем.

Іконка - Зазвичай представлена символом чашки кави, що стала відомим символом мови програмування Java.
 
Засновник - Мову Java розробила команда під керівництвом Джеймса Гослінга у компанії Sun Microsystems.

Специфіка - Специфіка Java полягає в її переносимості, безпеці, високій продуктивності та можливості працювати на різних платформах.

Сфери використання - Java використовується в різних сферах, таких як розробка програмного забезпечення для мобільних пристроїв, веб-програмування, вбудовані 
системи, фінансові програми та ігрова індустрія

Компанії, що використовують - Компанії, що використовують Java, включають Google, Amazon, Netflix, IBM, Oracle та багато інших, які використовують цю мову для 
розробки різноманітних продуктів та послуг
